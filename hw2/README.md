## Environment
- Linux Mint 19.1 (Ubuntu 18.04)
- `apt install flex bison libfl-dev gcc`
  - bison version: `3.0.4`
  - flex version: `2.6.4`
  - cc version: `7.4.0`

## Test

  - `yacc -d parser.y`
  - `cc -g -o parser lex.yy.c y.tab.c y.tab.h`
  - `./parser inut/{?}.p`

- Use script
  - `make test`
  - check files in `output` folder
